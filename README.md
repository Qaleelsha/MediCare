# MediCare – Hospital Appointment Booking Platform

MediCare is a full-stack MERN healthcare appointment booking platform where users can:

- Register and login securely
- Browse available doctors
- View specialization, timings, and consultation fees
- Book appointments online

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Vite
- Framer Motion

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication

### DevOps
- Docker
- Docker Compose
- Jenkins CI/CD

## Features
- JWT Authentication
- Doctor Listing
- Appointment Booking
- Responsive Professional UI
- Dockerized Full Stack Application
